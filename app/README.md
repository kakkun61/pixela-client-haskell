# Pixela client CLI

This is [Pixela](https://pixe.la/) client command line interface.

## Help

```
Î» pixela --help
Usage: pixela-cli [OPTION...] <COMMAND> [ARGS...]
  Pixela client

Options:
  -?     --help         display this help and exit
  -V     --version      output version information and exit
  -v[n]  --verbose[=n]  set verbosity level

Commands:
  create-user        create a new user
  update-token       update the token
  delete-user        delete the user
  create-graph       create a graph
  get-graphs         list up own graphs
  get-graph          get the graph SVG
  update-graph       update the graph
  delete-graph       delete the graph
  set-quantity       set the quantity of the specified date
  get-quantity       get the quantity of the specified date
  update-quantity    update the quantity of the specified date
  increment-quantity increment the quantity of the specified date
  decrement-quantity decrement the quantity of the specified date
  delete-quantity    update the quantity of the specified date
  create-webhook     create a new webhook
  get-webhooks       get webhooks
  invoke-webhook     invoke the webhook
  delete-webhook     delete the webhook
```
